# Сайт-резюме Георгия Воловикова

Это статический сайт-резюме, данные для которого загружаются из файла `resume.json`.

## Как запустить локально
Так как браузеры блокируют загрузку локальных JSON-файлов (ошибка CORS) при простом открытии `index.html`, вам нужно запустить любой локальный веб-сервер.

Если у вас установлен Python, выполните команду в папке проекта:
```bash
python3 -m http.server 8000
```
После чего откройте в браузере: [http://localhost:8000](http://localhost:8000)

## Как изменить данные
Вся информация находится в файле `resume.json`. Просто отредактируйте текст в этом файле, и сайт обновится автоматически при перезагрузке страницы.

## Как разместить в интернете бесплатно
1. **GitHub Pages:** Создайте репозиторий на GitHub, загрузите туда эти файлы и включите GitHub Pages в настройках репозитория.
2. **Vercel / Netlify:** Вы можете просто перетащить папку с проектом в окно загрузки на их сайтах (Vercel Drop или Netlify Drop), и они сразу дадут вам ссылку.
